# 8-30 新生训练

## 老师通知

入学第一天新生训练，主要内容：

* 按序号排队，保持安静。
* 听老师指令，按指令做好。请家长们给学生复习今天的指令。
  + 123，坐端正
  + 小嘴巴，不说话
  + 小耳朵，认真听
  + 立正1，2，请静息我静息
* 关于接送。
  + 家长务必按时或提前10分钟。
  + 请家长提醒学生放学时站在队伍中要向前看，找家长，接走时要和老师打招呼。
* 接送卡。建议套个外壳，接学生必须带上。
* 休息时间。入学后训练强度大，务必让孩子早睡早起。（今天课堂上有出现学生趴桌子打瞌睡的）
