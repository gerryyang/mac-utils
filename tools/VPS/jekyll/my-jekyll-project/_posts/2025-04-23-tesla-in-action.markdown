* 新能源上深圳牌，需要提交指标申请，因为指标是上牌的必要文件。

* 指标申请每月 5、15、25 号出结果，将通过短信通知结果。
* 客户登录[深圳小汽车增量调控管理信息系统](https://xqctk.jtys.sz.gov.cn/)，选择：增量 -> 个人 -> 省政务网入口 -> 微信扫码登录 -> 填写个人小汽车增量指标申请表 -> 选择申请类型纯电动小汽车。

### Tesla Model 3 历史价格走势

Tesla Model 3自2016年发布以来，经历了多次价格调整和改款升级，其价格策略和产品迭代反映了市场竞争、成本控制及技术革新的动态变化。以下是近十年的核心总结：

* 早期阶段（2017-2020年）：承诺与争议
    + 2017年马斯克承诺推出3.5万美元基础版Model 3，但早期实际售价偏高。例如，2019年国产Model 3初始售价为35.58万元（含Autopilot），后因补贴政策调整逐步降至29.9万元（2020年1月）
    + 2020年4月，国产Model 3标准续航版售价上涨4500元至30.355万元，长续航版上涨5000元至34.405万元，引发消费者不满

* 中期阶段（2021-2023年）：价格战与焕新版推出
    + 2023年焕新版Model 3上市后，起售价从26.14万元逐步调整至24.59万元（2024年1月），终端优惠叠加后价格更低
    + 特斯拉频繁通过保险补贴、免息政策等变相降价，**例如2025年2月推出“史上最大优惠组合”：保险补贴8000元 + 5年0息政策，起售价降至22.75万元**

* 全球市场差异
    + 中国市场因本土化生产和竞争压力，成为全球售价最低地区之一。例如，2024年焕新版Model 3起售价24.59万元，远低于同期美国市场的37,830美元（约26.9万元）
    + 其他市场如加拿大、美国因成本或战略调整，曾多次涨价

> 价格波动主因：本土化生产（国产化率从30%升至100%）、供应链成本下降、补贴政策及市场竞争（如比亚迪、蔚来等）



### Tesla Model 3 改款升级情况

* 2020年中期改款：补齐短板
    + 热泵空调 + PTC系统：提升冬季续航效率，制热能耗降低最高80%
    + 舒适性改进：增加双层玻璃（降低噪音）、电尾门、双手机无线充电等

* 2023年硬件升级：核心性能提升
    + 电池容量增加：搭载75度磷酸铁锂电池，续航提升至约650公里（EPA标准）
    + 第四代电机（4D1/4D5）：优化能效和成本，扁线绕组技术降低生产复杂度
    + HW4.0自动驾驶硬件：支持更多摄像头和4D毫米波雷达，算力提升3-5倍，解决“幽灵刹车”问题

* 2024年设计革新：内外兼修
    + 外观调整：取消雾灯，采用狭长灯组和Roadster风格前脸，新增扰流板
    + 内饰简化：取消怀挡设计，换挡集成至屏幕滑动操作；后排增加娱乐屏，支持Steam游戏平台
    + 制造工艺：采用后一体压铸车身，减少零部件数量，降低成本


> 改款逻辑：初期聚焦补齐短板（如噪音、续航），中期转向自动驾驶和制造效率，后期强化用户体验（娱乐功能、设计语言）
